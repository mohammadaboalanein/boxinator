# Boxinator

## To run server
Create schema in your database and call it Boxinator
Create in Boxinator a table with name box

Go to server dir by running:

### `cd server`

Then you can run:

### `node index.js`

## To run client 

Open client dir by running:
### `cd client`

And then you can run:

### `yarn start`

Or

### `npm start`


## To run test

Open client dir by running:
### `cd client`

And then you can run:

### `npm test`
